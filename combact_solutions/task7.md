# Simple Chatbot using LangGraph

## Environment Setup

### Requirements

* Python 3.10 or higher
* pip

### Install Dependencies

Install required packages using:

```bash
pip install -r requirements.txt
```

No API keys or external services are required.



## How to Run the Chatbot

Run the chatbot script from the terminal:

```bash
python simple_interactive_chatbot.py
```

The chatbot will start and wait for user input.

To exit the chatbot, type:

```text
exit
```

or

```text
quit
```



## Example Input / Output

```
 Simple Chatbot started. Type 'exit' to quit.

User: hi
Assistant: Hi! How can I help you today?

User: i am learning langgraph
Assistant: Okay, tell me more.

User: exit
 Chat ended.
```

This example shows that the chatbot runs interactively and maintains conversational context.
